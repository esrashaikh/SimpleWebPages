When was the start of the web and web design?
In 1989, whilst working at CERN, Tim Berners-Lee proposed to create a global hypertext project,
which later became known as the World Wide Web. From 1991 to 1993 the World Wide Web was born.
Text-only HTML pages could be viewed using a simple line-mode web browser.
